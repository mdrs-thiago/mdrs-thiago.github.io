Sobre

Olá! Meu nome é Thiago Medeiros, e sou professor do Departamento de Engenharia de Sistemas e Computação. Tenho formação de técnico em Telecomunicações (CEFET-RJ, 2012), graduação em Engenharia Elétrica com ênfase em Sistemas Eletrônicos (UERJ, 2018), mestrado em Engenharia Elétrica - Métodos de Apoio à Decisão (PUC-Rio, 2021) e doutorando em Engenharia Elétrica - Métodos de Apoio à Decisão (PUC-Rio, 2025). Atualmente trabalho na área de Deep Learning, mais especificamente utilizando modelos LLMs e em aplicações como detecção de amostras fora da distribuição. Quando possível, também trabalho em outras aplicações, desde projetos de ciência de dados até circuitos eletrônicos. :-)

Este portal foi criado para compartilhar conhecimentos através de aulas teóricas, práticas, artigos, projetos e atualizações. Aqui você encontrará conteúdos que vão desde conceitos básicos até aplicações avançadas, servindo como um centralizador para todos (mas não exclusivamente) os alunos da minha disciplina.

Projetos em Aberto

Aplicação de Detecção de Objetos/Segmentação Semântica usando câmera OAK-D

Desenvolver um modelo para detecção de objetos em tarefas a definir e aplicá-los em ambiente real.

Aplicação de LLMs para extração de informações

Desenvolver um sistema de LLM que seja capaz de extrair informações de um documento de interesse.

Exemplo: usar LLM para extrair informações de um prontuário médico.

Aplicação de LLMs para estruturação de documentos

Criar um sistema que, dado um documento com formas complexas (por exemplo, tabelas e figuras) retorne um documento que seja mais legível ao LLM.

Exemplo: usar LLM para transformar uma tabela do PDF em um formato tipo latex.

Explicabilidade em modelos de Visão Computacional

Avaliar técnicas de explicabilidade para tarefas de Deep Learning.

Exemplo: aplicação de explicabilidade para entender a região de importância para uma classe de detecção de câncer.

Aplicações de Deep Learning em Sensoriamento Remoto

Avaliar técnicas para classificação ou segmentação semântica em dados obtidos por satélites

Exemplo: Segmentação de desmatamento na Amazônia.

Uso de robótica móvel com visão computacional

Desenvolver sistemas para robótica inteligente usando modelos de visão computacional. As aplicações podem abranger tanto robôs terrestres (e.g., carrinho) ou drones.

Exemplo: Robô móvel inteligente para inspeção de um ambiente.

Desenvolvimento de ambientes para habilitar agentes autônomos

Desenvolver aplicações (um jogo ou um ambiente simulado) de interesse para habilitar o uso de modelos de aprendizado de máquina.

Exemplo: Desenvolvimento de um jogo estilo Flappy Bird que possibilite o feedback do ambiente e a ação de um agente autônomo.

Análise de Dados e Aprendizado de Máquina com Dados Abertos

Utilizar dados públicos para desenvolver aplicações de análise de dados (e.g., construção de dashboards) e uso de modelos de aprendizado de máquina para descoberta de padrões.

Exemplo: Criação de um projeto integrado de Análise de Dados e descoberta de padrões em informações de ações do mercado de petróleo no Brasil.
